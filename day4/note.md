## 一、函数基础

### 1. 函数定义与调用

- 函数是完成特定功能的代码块，定义后可以重复调用
- 语法：

```python
def 函数名(参数列表):
    """函数文档字符串：说明函数的功能、参数、返回值"""
    函数体代码
    return 返回值
```

- 调用：`函数名(实参列表)`

### 2. 函数返回值

- `return` 语句用于返回函数的执行结果，函数执行到 `return` 会立即结束
- 一个函数可以没有 `return` 语句，此时默认返回 `None`
- 一个函数可以返回多个值，多个值会被打包成元组返回

### 3. 文档字符串

- 函数定义后第一行的三引号字符串，用于说明函数的功能、参数、返回值
- 可以通过 `help(函数名)` 或 `函数名.__doc__` 查看文档字符串
- 良好的文档字符串是代码可读性和可维护性的重要保障

## 二、函数参数体系（核心重点）

### 1. 位置参数

- 最基础的参数类型，调用时必须按定义顺序传入相同数量的实参
- 实参与形参按位置一一对应

### 2. 默认参数

- 定义函数时给参数指定默认值，调用时如果没有传入该参数，会使用默认值
- 默认参数必须放在位置参数后面
- **重要注意点**：默认参数必须使用不可变类型（int、str、tuple 等），不能使用可变类型（list、dict 等），否则会出现意想不到的结果

### 3. 关键字参数

- 调用函数时，通过 `参数名=值` 的形式传入参数
- 关键字参数可以不按定义顺序传入，提高代码可读性
- 关键字参数必须放在位置参数后面

### 4. 不定长位置参数 `*args`

- 用于接收任意数量的位置参数，这些参数会被打包成一个元组
- `*args` 必须放在位置参数和默认参数后面

### 5. 不定长关键字参数 `**kwargs`

- 用于接收任意数量的关键字参数，这些参数会被打包成一个字典
- `**kwargs` 必须放在所有参数的最后

### 6. 参数组合使用

- 所有参数类型可以组合使用，但必须严格按照顺序：位置参数 → 默认参数 → *args → 关键字参数 → **kwargs

## 三、函数作用域

### 1. 局部变量与全局变量

- 局部变量：在函数内部定义的变量，只能在函数内部访问
- 全局变量：在函数外部定义的变量，可以在整个模块中访问

### 2. global 关键字

- 如果想在函数内部修改全局变量的值，必须使用 `global` 关键字声明
- 使用 `global` 声明，在函数内部给全局变量赋值会被认为是定义了一个同名的局部变量

### 3. nonlocal 关键字

- 用于在嵌套函数内部修改外层函数的局部变量
- 不能用于修改全局变量

## 四、匿名函数 lambda

- lambda 是一种简洁的匿名函数定义方式，用于创建简单的、一次性使用的函数
- 语法：`lambda 参数列表: 表达式`
- lambda 函数只能有一个表达式，表达式的结果就是返回值
- 常用于作为高阶函数的参数，简化代码

## 五、三大高阶函数（AI 数据处理高频使用）

- 高阶函数：接收函数作为参数，或者返回函数的函数
- Python 内置了三个常用的高阶函数：map、filter、reduce，在数据处理中非常常用

### 1. map 函数

- 作用：将一个函数应用于可迭代对象的每个元素，返回一个新的迭代器
- 语法：`map(函数, 可迭代对象1, 可迭代对象2, ...)`

### 2. filter 函数

- 作用：过滤可迭代对象中满足条件的元素，返回一个新的迭代器
- 语法：`filter(函数, 可迭代对象)`
- 函数返回布尔值，返回 True 的元素会被保留

### 3. reduce 函数

- 作用：对可迭代对象的元素进行累积操作，返回一个单一的值
- 语法：`reduce(函数, 可迭代对象[, 初始值])`
- 函数必须接收两个参数，reduce 会将前一次的计算结果作为下一次的第一个参数
- 注意：reduce 在 Python3 中被移到了 functools 模块，需要先导入

## 六、闭包基础

- 闭包是指嵌套函数中，内层函数引用了外层函数的变量，并且外层函数返回了内层函数
- 闭包可以保留外层函数的变量状态，即使外层函数已经执行完毕
- 闭包是装饰器的基础（Day12 会深入学习）

```python
def outer(x):
    def inner(y):
        return x + y

    return inner


# 调用外层函数，得到内层函数
add5 = outer(5)
add10 = outer(10)

# 调用内层函数
print(add5(3))  # 输出：8
print(add10(3))  # 输出：13
```

## 七、今日易错点总结

1. **默认参数使用可变类型**：导致默认值被复用，出现意想不到的结果
2. **参数顺序错误**：不按照 "位置→默认→*args→关键字→**kwargs" 的顺序定义参数
3. **混淆 global 和 nonlocal**：global 用于修改全局变量，nonlocal 用于修改外层函数局部变量
4. **lambda 函数只能有一个表达式**：不能包含循环、分支等复杂逻辑
5. **reduce 函数需要导入**：Python3 中 reduce 在 functools 模块，需要先导入
6. **函数返回多个值时忘记解包**：多返回值会被打包成元组，需要解包才能分别使用
7. **在函数内部修改全局变量忘记加 global**：会被认为是定义了同名局部变量
8. **文档字符串缺失**：降低代码可读性和可维护性
